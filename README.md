# Enter-the-Fold
A 2D laundry store management game built in Godot 4

Enter the Fold is a standalone 2D laundry store management game built in Godot 4.
Customers walk into your store, drop off their laundry, and return later to pick it up. You type words and letters to perform every action — picking up baskets, loading machines, folding clothes, serving customers. The faster and more accurately you type, the more customers you serve, the more money you make, and the further you progress.
The name is a double meaning: folding laundry and pressing the Enter key — the last keystroke before a customer walks out satisfied.
